# 帮助:格式标准

> [!WARNING] 特别强调
> 请各编辑者在编辑笔记时注意术语！“条目”是用于百科全书之内容之称呼，而“词条”用于辞典，我们的工作则可以描述为“笔记”。

> [!CAUTION] 关于模板
> 模板的作用是为了突出你要表达的提醒、警告或其他用途，应当尽量保持简洁直观。对一个界面使用过多、过于繁杂的模板弊大于利，很有可能导致模板滥用或页面混乱。

如果您是第一次编辑本网站的笔记，请花点时间阅读**格式标准**和**其他笔记**，这可以对您的格式起到很大帮助，同时糟糕的格式将会被行政员**修改或删除**。

另外，请在每次创建新页面时这样做 -- 貓塚千明如是说
```
---
tags:
  - 你的主题
  - 你的主题/格式标细分领域
---
```

## Markdown 语法
### 1. 标题（Headers）

```markdown
# 一级标题 (H1)
## 二级标题 (H2)
### 三级标题 (H3)
#### 四级标题 (H4)
##### 五级标题 (H5)
###### 六级标题 (H6)
```

**演示效果：**
# 一级标题 (H1)
## 二级标题 (H2)
### 三级标题 (H3)
#### 四级标题 (H4)
##### 五级标题 (H5)
###### 六级标题 (H6)

---

### 2. 强调文本

```markdown
**这是加粗文字**
*这是斜体文字*
***这是加粗斜体***
~~这是删除线~~
```

**演示效果：**
**这是加粗文字**  
*这是斜体文字*  
***这是加粗斜体***  
~~这是删除线~~

---

### 3. 列表

#### 无序列表
```markdown
- 项目1
- 项目2
  - 子项目1
  - 子项目2
* 项目3
+ 项目4
```

**演示效果：**
- 项目1
- 项目2
  - 子项目1
  - 子项目2
* 项目3
+ 项目4

#### 有序列表
```markdown
1. 第一项
2. 第二项
3. 第三项
   4. 子项1
   5. 子项2
```

**演示效果：**
1. 第一项
2. 第二项
3. 第三项
   4. 子项1
   5. 子项2

---

### 4. 链接与图片

#### 链接
```markdown
[普通链接](https://www.example.com)
[带标题的链接](https://www.example.com "鼠标悬停显示的文字")
[引用式链接][1]

[1]: https://www.example.com
```

**演示效果：**
[普通链接](https://www.example.com)  
[带标题的链接](https://www.example.com "鼠标悬停显示的文字")

#### 图片
```markdown

```

**演示效果：**
![图片替代文字](https://picsum.photos/600/300 "图片标题")
